# hardware
Open Hardware Air Pollution sensor

## housing
KRADEX Z56JPH housing is selected. Drawings and  3D models are available at [manufacturers site](https://www.kradex.com.pl/shop?product=z56).

## 3D model
To see proper 3D model, copy files from \lib\packages3d\ to KiCad KISYS3DMOD directory (i.e. C:\Program Files\KiCad\share\kicad\modules\packages3d on Windows)
